// 03 — Kebijakan Privasi

Privasi & Keamanan Data

Transparan & Bertanggung Jawab

Komitmen Kami Terhadap Data Anda

ASN Master dirancang dengan prinsip privacy-by-design. Aplikasi tidak memerlukan akun atau login — tidak ada data identitas yang dikumpulkan. Berikut adalah penjelasan transparan tentang bagaimana data Anda dikelola.

📸Kamera
🔒SSL/HTTPS
🤖Gemini AI
📄Laporan
🗑️Data Terhapus
PASAL 01
📸
Pengumpulan Data Foto

Aplikasi hanya mengambil foto kegiatan ketika fitur analisis AI secara eksplisit diaktifkan oleh pengguna. Tidak ada kamera yang berjalan di latar belakang atau pengambilan foto otomatis.

Zero Background Access: Kamera hanya aktif atas permintaan langsung pengguna. Foto tidak pernah diakses di luar sesi analisis yang sedang berlangsung.
PASAL 02
🗑️
Retensi Data Server

Foto yang dikirimkan ke Gemini API untuk keperluan analisis tidak disimpan secara permanen di server kami. Setelah proses analisis selesai dan hasil dikembalikan ke aplikasi, foto tersebut dihapus dari pipeline pemrosesan.

Non-Persistent Storage: Kami menerapkan kebijakan zero-retention untuk semua gambar yang diproses oleh sistem AI kami.
PASAL 03
🔐
Keamanan Transmisi

Seluruh komunikasi antara aplikasi dan server AI dilindungi menggunakan enkripsi SSL/TLS (HTTPS). Ini memastikan bahwa foto dan data laporan tidak dapat disadap oleh pihak ketiga selama proses pengiriman.

End-to-End Encrypted: Protokol HTTPS dengan sertifikat SSL standar industri diterapkan pada semua endpoint API.
PASAL 04
💾
Penyimpanan Lokal (Room DB)

Riwayat laporan kegiatan disimpan secara lokal di perangkat Anda menggunakan Room Database — database terenkripsi yang merupakan bagian dari ekosistem Android Jetpack. Data ini sepenuhnya berada dalam kendali pengguna.

On-Device Only: Riwayat laporan dapat dihapus kapan saja melalui menu Riwayat di aplikasi, tanpa perlu koneksi internet.
PASAL 05
👤
Tanpa Akun & Identitas

ASN Master tidak memerlukan registrasi atau login. Tidak ada nama, email, nomor telepon, atau identitas pribadi lain yang dikumpulkan oleh sistem kami. Penggunaan aplikasi bersifat anonim secara struktural.

No PII Collection: Tidak ada Personally Identifiable Information yang pernah melewati server kami.
PASAL 06
🔄
Layanan Pihak Ketiga

Aplikasi menggunakan Google Gemini API sebagai mesin AI. Pemrosesan data oleh Gemini API tunduk pada kebijakan privasi Google. Kami merekomendasikan pengguna untuk meninjau kebijakan privasi Google terkait penggunaan API tersebut.

Third-Party: Google Gemini API — policies.google.com/privacy

Hak Pengguna

Sebagai pengguna ASN Master, Anda memiliki kontrol penuh atas data lokal Anda.

  • 🗑️ Hapus Riwayat: Hapus semua laporan tersimpan kapan saja melalui menu Riwayat di aplikasi.
  • 📴 Cabut Izin Kamera: Nonaktifkan izin kamera melalui pengaturan perangkat Anda untuk mencegah akses foto.
  • 📵 Mode Offline: Fitur non-AI tetap berfungsi tanpa koneksi internet — tidak ada data yang dikirim.
  • 🔍 Transparansi: Kebijakan privasi ini tersedia secara publik dan diperbarui setiap kali ada perubahan signifikan.
