// RINGKASAN

Kebijakan Privasi ini menjelaskan bagaimana DocuScan AI ("kami", "kita", atau "aplikasi") mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda saat menggunakan layanan kami. Dengan menggunakan aplikasi ini, Anda menyetujui praktik yang dijelaskan dalam kebijakan ini.

// 01

🔭 Pendahuluan & Ruang Lingkup

DocuScan AI adalah aplikasi Android yang dikembangkan oleh El-Salam Technology, berdomisili di Indonesia. Kebijakan Privasi ini berlaku untuk semua pengguna aplikasi kami di seluruh dunia.

Kami berkomitmen untuk melindungi privasi Anda dan memastikan data pribadi Anda ditangani dengan aman, transparan, dan sesuai dengan peraturan yang berlaku, termasuk Undang-Undang Perlindungan Data Pribadi (UU PDP) Indonesia, GDPR (untuk pengguna Eropa), dan kebijakan Google Play.

// YURISDIKSI

Kebijakan ini tunduk pada hukum Republik Indonesia. Pengguna dari wilayah lain juga dilindungi oleh hak-hak yang diberikan oleh regulasi lokal yang berlaku di wilayah mereka.

// 02

📡 Data yang Kami Kumpulkan

Kami mengumpulkan beberapa jenis informasi untuk menyediakan dan meningkatkan layanan kami:

DATA YANG DIKUMPULKAN OTOMATIS

  • Jenis perangkat, model, dan sistem operasi Android.
  • Versi aplikasi yang digunakan.
  • Alamat IP dan data jaringan (Wi-Fi / seluler).
  • Log aktivitas, crash report, dan data diagnostik melalui Firebase Crashlytics.
  • Token Firebase / FCM untuk notifikasi push.

DATA YANG DIKUMPULKAN OTOMATIS (PIHAK KETIGA)

  • Informasi perangkat (Model, Versi OS Android).
  • ID Iklan (Google Advertising ID) untuk keperluan penayangan iklan melalui Google AdMob.
  • Log aktivitas teknis dan data diagnostik (Crashlytics) untuk perbaikan aplikasi.

DATA BERBASIS IZIN

  • Kamera — Digunakan untuk mengambil gambar dokumen yang akan dianalisis secara cerdas.
  • Penyimpanan — Digunakan untuk membaca file gambar/PDF dari galeri dan menyimpan hasil ekspor PDF ke perangkat Anda sebelum dibagikan ke WhatsApp.
  • Internet — Digunakan untuk memproses analisis dokumen menggunakan teknologi AI (Gemini) dan menampilkan iklan.
// PERHATIAN

Kami tidak pernah mengumpulkan nama, alamat email, atau nomor telepon secara langsung karena aplikasi ini dapat digunakan tanpa registrasi akun (anonim). Kami juga tidak pernah mengumpulkan data biometrik atau data sensitif lainnya tanpa persetujuan eksplisit Anda.

// 03

⚙️ Bagaimana Kami Menggunakan Data Anda

Kami menggunakan informasi yang dikumpulkan untuk tujuan-tujuan berikut:

  • Menyediakan fitur pemindaian dan analisis dokumen berbasis AI.
  • Memproses konversi dokumen ke format PDF secara lokal maupun berbasis cloud.
  • Meningkatkan akurasi analisis AI dan performa aplikasi.
  • Menampilkan iklan yang relevan melalui layanan pihak ketiga (AdMob).
  • Mencegah penyalahgunaan layanan dan mematuhi kewajiban hukum.
// 04

🔗 Berbagi & Pengungkapan Data

Kami tidak menjual data pribadi Anda. Data Anda dibagikan secara terbatas hanya kepada:

  • Layanan AI (OpenRouter/Gemini) — Gambar dokumen dikirimkan ke server API untuk diproses analisisnya secara sementara dan tidak disimpan secara permanen oleh pengembang.
  • Google Firebase — Untuk analitik penggunaan dan laporan kerusakan aplikasi.
  • Google AdMob — Untuk penayangan iklan di dalam aplikasi. Setiap layanan memiliki kebijakan privasi masing-masing yang terikat pada perlindungan data global.
// 05

🛡️ Keamanan Data

Keamanan data Anda adalah prioritas kami:

  • Pengiriman gambar untuk analisis dilakukan melalui jalur terenkripsi HTTPS/TLS.
  • Kami meminimalkan penyimpanan data di server; sebagian besar proses (seperti pembuatan PDF) dilakukan langsung di perangkat Anda.
// 06

⚖️ Hak-Hak Pengguna

Anda memiliki hak untuk:

  • Menghentikan pengumpulan data dengan menghapus aplikasi.
  • Mengatur izin Kamera dan Penyimpanan melalui pengaturan sistem Android kapan saja.
  • Menghubungi kami jika memiliki kekhawatiran terkait data yang diproses selama analisis dokumen.
// 07

👶 Perlindungan Anak

Aplikasi kami tidak ditujukan untuk anak-anak di bawah usia 13 tahun (atau usia minimum yang berlaku di yurisdiksi Anda). Jika Anda adalah orang tua dan mengetahui anak Anda memberikan informasi kepada kami, silakan hubungi kami untuk penghapusan data.

// 08

🌐 Layanan Pihak Ketiga

Aplikasi kami mungkin mengintegrasikan layanan dari pihak ketiga. Setiap layanan beroperasi di bawah kebijakan privasi mereka sendiri. Kami mendorong Anda untuk membaca kebijakan pihak ketiga yang kami gunakan:

  • Google Play Services — policies.google.com/privacy
  • Google AdMob — support.google.com/admob
  • Google Firebase — firebase.google.com/support/privacy
  • OpenRouter (AI Provider) — openrouter.ai/privacy

Kami tidak bertanggung jawab atas praktik privasi pihak ketiga tersebut. Penggunaan layanan mereka tunduk pada syarat dan ketentuan masing-masing.
